Transmisión Óptica (Pantalla ↔ Cámara)

Consejo: Para pruebas, pon Bit Duration 150–300 ms. Mantener pantalla cerca del receptor (20–60 cm).

La detección automática intenta encontrar el preámbulo. Si falla, empezá recepción justo antes de iniciar la transmisión.

Modo Transmitir: pantalla de datos
Notas: Este prototipo demuestra la transmisión óptica básica. Para mayor velocidad/robustez usar grillas, RGB, FEC y manejo de rolling-shutter.
